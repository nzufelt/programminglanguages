How to contribute.

This project is nowhere near finished and is supposed to grow with time. That is why I need your help, no matter your skill level.

If you spot any mistakes, have improvements or want to add information about a programming languages, just submit a pull request and I will take a look at it. Pull requests are greatly appreciated!
